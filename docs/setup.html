getting started

getting started with nexus is fairly simple to do

it doesnt involve doing to much but it does require a linux system

being comfortable with the commandline would also help but isnt required

setting Nexus up!

git clone https://github.com/JohnTeBrown/Nexus-Target-Verifier.git

clone the repo like above and follow the steps below


sudo apt-get install cmake -y

then run the commands below to finish the installation process

sudo cmake CMakeLists.txt

and there you have it... you have installed and setup nexus

a extracted version of Nexus should be on your Desktop.
