Unsupported OS

Nexus only works on certain supported LINUX/UNIX distros

it will not function or work on Microsoft Windows or Apple Mac OSX.

Nexus supports:

  1. Debian/Debian based distros
  2. Arch/Arch based distros
  3. FreeBSD
  4. Solaris

if your OS is either based or the following listed above you will be met with a error upon the Nexus package build stage

please stay up to date with news as we may add support for future operating systems or Linux Distros


Package's not found

if Nexus is unable to find the packages on your system rerun the program

during the setup phase nexus installs the packages but will still give a error stating the packages are not found

rerunning the program should work because the formerly missing packages will be installed a ready to be used

if your package manager for some reason cannot find the packages in it's repo's

then please find other means to add repo's that do carry the required packages too your package manager

if for some reason you are unable to add repo's that can be read and used by your packaage manager

then please build the required packages from scratch

required package list
  1. Lua
  2. Cmake
  3. Python
