Troubleshooting Nexus

first step to troubleshooting, if you are having issues with Nexus please open up a github issue @Nexus Issues

issues
Unsupported OS

Nexus only works on certain supported LINUX/UNIX distros

it will not function or work on Microsoft Windows or Apple Mac OSX.

Nexus supports:

  1. Debian/Debian based distros
  2. Arch/Arch based distros
  3. FreeBSD
  4. Solaris

if your OS is either based or the following listed above you will be met with a error upon the Nexus package build stage

please stay up to date with news as we may add support for future operating systems or Linux Distros


Package's not found

if Nexus is unable to find the packages on your system rerun the program

during the setup phase nexus installs the packages but will still give a error stating the packages are not found

rerunning the program should work because the formerly missing packages will be installed a ready to be used

if your package manager for some reason cannot find the packages in it's repo's

then please find other means to add repo's that do carry the required packages too your package manager

if for some reason you are unable to add repo's that can be read and used by your packaage manager

then please build the required packages from scratch

required package list
  1. Lua
  2. Cmake
  3. Python

Cmake install error "Nebulae"

Cmake error nebulae is a error that occurs when your cmake package is either corrupted or missing essential files to work

to fix the error please first attempt to reinstall cmake via your package manager

if your using apt or dpkg on debian based systems attempt to fix broken or missing packages

if this method of fixing error code nebulae fails please make a github issue for further support

Error code example
| Cmake install failed ;; "Neblae" |

Lua version error

Lua is more of a backup package and does not quite serve a purpose yet

if you encounter a error that mentions conflicts with the lua package then please uninstall Lua and clear your package cache

after uninstalling lua please make a github issue so we can look into ways of preventing this common package conflict

please re-run the program once the package is gone and you have cleared it's cache files.


Clang failed for X reason

if clang fails to compile and read the C, C++ files then please make a github issue that includes the log file

once you have made a github issue please wait for a response and work with contributors or junior developers as needed

if your issue is not resolved your issue ticket on github will be marked with P2 Unsolved as it's status

this means your issue will become a slightly higher priority and we will mark the issue as solved once we resolve the error/bug.


security & risk concerns

if you are concerned about the security of your system while using Nexus and Nexus related tools or products

then please refer to the SECURITY.md on github

keep your linux kernel up to date to protect against potential vulnerabilties and exploits

refrain from running dangerouse scripts or running root level commands via the web console tab in the nexus panel

refrain from installing or running scripts or root level commands in the container shell "shell.sh"

the container shell connects directly to the devilbox backend which is the engine that gives Nexus life.
